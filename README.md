# value-bets

Betting picks for matches with kickoff times between 00:00 UTC and 11:59 UTC

## Daily Notification Schedule
- **Email sent at:** 00:00 UTC (Midnight)
- **Pick kickoff timeline:** 00:00 - 11:59 UTC
- **Frequency:** Daily

## How to Track Your Picks
1. Check this repository for daily betting picks
2. Picks are filtered to show only matches kicking off between midnight and noon UTC
3. Receive email notifications at 00:00 UTC each day

## Repository Structure
- `picks/` - Daily pick records
- `results/` - Tracking bet outcomes
- `.github/workflows/` - Automation scripts
